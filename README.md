**Description:**  
WeatherPulse lets users check real-time weather for any city. It fetches data from OpenWeather API and displays temperature, humidity, and weather conditions with a clean, modern interface.  

**Features:**  
- Search weather by city  
- Display temperature, humidity, and weather status  
- Responsive layout using Tailwind CSS  
- Modern UI with clean design
